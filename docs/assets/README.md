## How to capture

1. Run the app: `python app.py`
2. Upload `data/feedback.csv` to get a populated session
3. Navigate to each page and capture at 1440×900 (desktop) and 390×844 (mobile)

Recommended tools: browser DevTools device emulation, or [Playwright](https://playwright.dev/) for automated screenshots.
